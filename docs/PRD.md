# PRD: TubeWiki — YouTube 요약 위키 생성기

## 목표
유튜브 링크 하나로 영상 전체 내용을 과하지 않게 구조적으로 요약하고, 제목·썸네일이 실제 내용과 일치하는지 검증하며, LLM 위키의 소스로 쓸 정보 밀도 높은 MD 문서를 생성한다.

## 사용자
- 영상을 끝까지 보지 않고도 전체 맥락을 정확히 파악하려는 사람
- 유튜브 내용을 지식 베이스(LLM 위키)의 소스 문서로 축적하려는 사람

## 핵심 기능
1. 유튜브 URL 입력 → 자막 + 메타데이터(제목·채널·썸네일·길이) 수집
2. 과잉 압축하지 않는 구조적 요약 — 전체 흐름을 파악할 수 있는 개요 + 섹션별 상세
3. 제목/썸네일 vs 실제 내용 일치 검증 — 낚시 감지 (일치 / 부분일치 / 불일치 + 근거)
4. 정보 밀도 높은 MD 문서 생성 및 다운로드 — LLM 위키 소스용 (인용·수치·인물·용어 보존)

## MVP 제외 사항
- 자막 없는 영상의 음성 인식(Whisper) 폴백 — 자막 없으면 명확한 안내 메시지로 종료
- 요약 히스토리 저장 / DB
- 로그인 / 사용자 계정
- 다국어 UI (한국어 고정, 출력 문서도 항상 한국어)

## 디자인
- 라이트 테마 기본, 무채색(neutral) + 검증 결과에만 시맨틱 색(초록/주황/빨강)
- 미니멀 "도구" 느낌. 마케팅 랜딩이 아니라 매일 쓰는 유틸리티
- AI 슬롭 안티패턴 회피 (UI_GUIDE.md 준수)
